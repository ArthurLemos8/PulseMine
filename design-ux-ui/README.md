# design-ux-ui
Definição visual inicial do projeto PulseMine.

Inclui:
- Paleta principal de cores do sistema
- Tons de azul para identidade visual
- Azul escuro e preto para elementos principais
- Branco e cinzas para fundos e componentes
- Cor de alerta para estados críticos

A paleta será utilizada como base para as telas, dashboard e componentes da interface.
